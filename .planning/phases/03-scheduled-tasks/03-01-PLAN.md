Create `src/scheduler/scheduler-manager.ts` implementing the full scheduler engine. Key behaviors:

**Imports and state:**
```typescript
import { Cron } from "croner";
import { randomUUID } from "node:crypto";
import type Database from "better-sqlite3";
import type { ScheduledTaskRow, SchedulerDeps, TaskType, TaskStatus } from "./types.js";
import { log } from "../logger.js";
```

Module-level `activeJobs` Map<string, Cron> to track running croner instances.
Module-level `taskQueue` array and `isExecuting` boolean for sequential execution.
Module-level `deps` variable of type `SchedulerDeps | null` set by `startScheduler`.

**Prepared statements** (created once in startScheduler):
- `selectActive`: `SELECT * FROM scheduled_tasks WHERE status = 'active'`
- `selectByUser`: `SELECT * FROM scheduled_tasks WHERE user_id = ? ORDER BY created_at DESC`
- `selectById`: `SELECT * FROM scheduled_tasks WHERE id = ?`
- `insertTask`: INSERT into scheduled_tasks with all fields
- `updateStatus`: UPDATE status + updated_at WHERE id = ?
- `updateAfterRun`: UPDATE last_run_at, run_count = run_count + 1, last_error = NULL, retry_after = NULL WHERE id = ?
- `updateError`: UPDATE last_error, retry_after WHERE id = ?
- `deleteTask`: DELETE FROM scheduled_tasks WHERE id = ?

**startScheduler(schedulerDeps: SchedulerDeps): void**
- Store deps in module-level variable
- Create prepared statements
- Load all active tasks via selectActive
- For each row, call `registerJob(row)`
- Log how many tasks loaded: `log("info", "scheduler", "Scheduler started", { tasksLoaded: rows.length })`

**registerJob(task: ScheduledTaskRow): void**
- If activeJobs has task.id, stop existing job first
- Create new Cron with:
  - expression: task.cron_expression
  - options: `{ name: task.id, timezone: task.timezone === "local" ? undefined : task.timezone, protect: true, paused: task.status === "paused" }`
  - callback: `async () => { enqueueExecution(task); }`
- Store in activeJobs Map

**enqueueExecution(task: ScheduledTaskRow): void**
- Push task to taskQueue
- If !isExecuting, call processQueue()

**async processQueue(): Promise<void>**
- Set isExecuting = true
- While taskQueue.length > 0, shift and call executeTask
- Set isExecuting = false

**async executeTask(task: ScheduledTaskRow): Promise<void>**
- Re-read task from DB (selectById) to get fresh status â€” skip if status !== 'active'
- try block:
  - If task.type === "reminder":
    - `await deps.sendMessage(task.user_id, "đź”” Recordatorio: " + task.prompt)`
  - Else (task, briefing, pr-monitor):
    - `const sessionId = deps.memoryManager.resolveSession(task.user_id, "scheduler", 0)`
    - `await deps.runAgent({ userId: task.user_id, userName: "scheduler", channelId: "scheduler", sessionId, userMessage: task.prompt }, deps.llm, deps.toolRegistry, deps.memoryManager, deps.soulContent, deps.maxIterations)`
  - After success: `updateAfterRun.run(task.id)`
  - Check if one-shot (use Cron.nextRun() â€” if null after run, it's done):
    - `const job = activeJobs.get(task.id); if (job && job.nextRun() === null) { updateStatus.run("completed", task.id); job.stop(); activeJobs.delete(task.id); }`
- catch block: call handleTaskError(task, err)

**async handleTaskError(task: ScheduledTaskRow, err: unknown): Promise<void>**
- Get errorMsg from err
- Check if this is already a retry (task.retry_after is set and past):
  - If retry_after was set (this is second failure): `updateError.run(errorMsg, null, task.id)` â€” clear retry_after, leave active for next scheduled run
  - Send: `await deps.sendMessage(task.user_id, "âťŚ Tarea \"${task.name}\" fallĂł de nuevo: ${errorMsg}\nEsperando prĂłxima ejecuciĂłn programada.")`
- Else (first failure):
  - `updateError.run(errorMsg, new Date(Date.now() + 5 * 60 * 1000).toISOString(), task.id)`
  - Send: `await deps.sendMessage(task.user_id, "âťŚ Tarea \"${task.name}\" fallĂł: ${errorMsg}\nReintentando en 5 minutos...")`
  - Schedule retry with setTimeout: `setTimeout(() => { enqueueExecution(task); }, 5 * 60 * 1000)`
- Log the error: `log("error", "scheduler", "Task execution failed", { taskId: task.id, taskName: task.name, error: errorMsg })`

**createTask(params: { userId, name, type, cronExpression, prompt, timezone?, preApproved? }): ScheduledTaskRow**
- Generate id with randomUUID()
- Insert into DB via insertTask
- Read back the row via selectById
- Call registerJob on the new row
- Log: `log("info", "scheduler", "Task created", { taskId: id, taskName: params.name, type: params.type })`
- Return the row

**deleteTask(taskId: string): boolean**
- Stop and remove from activeJobs if present
- Run deleteTask statement
- Return changes > 0

**pauseTask(taskId: string): boolean**
- updateStatus.run("paused", taskId)
- If activeJobs has taskId: activeJobs.get(taskId).pause()
- Return true if row found

**resumeTask(taskId: string): boolean**
- updateStatus.run("active", taskId)
- If activeJobs has taskId: activeJobs.get(taskId).resume()
- Return true if row found

**listTasks(userId: string): ScheduledTaskRow[]**
- Return selectByUser.all(userId) cast to ScheduledTaskRow[]

**getNextRun(taskId: string): Date | null**
- If activeJobs has taskId: return activeJobs.get(taskId).nextRun()
- Return null

**stopAll(): void**
- For each job in activeJobs values: job.stop()
- activeJobs.clear()
- Log: `log("info", "scheduler", "All scheduler jobs stopped")`

Export all public functions: startScheduler, createTask, deleteTask, pauseTask, resumeTask, listTasks, getNextRun, stopAll.
